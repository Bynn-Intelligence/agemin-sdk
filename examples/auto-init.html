Auto-Initialization Example

This example demonstrates automatic SDK initialization using data attributes on the script tag. The SDK is configured without any JavaScript code!


<script src="agemin-sdk.js"
  data-agemin-asset-id="asset_5b08b274353b92f4"
  data-agemin-session-id="server-generated-session-id"
  data-agemin-theme="auto"
  data-agemin-locale="en"
  data-agemin-debug="true">
</script>
✨
Zero configuration required
🚀
Automatic initialization on page load
🎯
Auto-binds to elements with data-agemin-trigger
🔧
Configure everything via HTML attributes
